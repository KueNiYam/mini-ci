# Mini CI 프로젝트 규칙

- 사용자에게 공개되거나 온보딩에 쓰이는 문서는 영어를 기본 본문으로 작성하고, 한국어 번역은 바로 아래 blockquote로 제공합니다.
- 적용 대상은 `README.md`, `docs/` 아래 문서처럼 사용자가 직접 읽는 문서입니다.
- 새로 작성하거나 수정하는 코드 주석과 TSDoc은 병기하지 않고 한국어로 작성합니다.
- `TODO.md`처럼 내부 작업 추적용 문서는 병기하지 않아도 됩니다.
- 제목, 명령어, API 경로, 파일명, 코드 식별자처럼 그대로 실행하거나 참조해야 하는 값은 번역하지 않고 영어 원문을 유지합니다.
- 짧은 용어형 항목은 필요할 때 `English / 한국어` 형태를 사용할 수 있습니다.
